# Opticanew

Opticanew — мобильный исследовательский сайт про скрытые механизмы давления в тексте.

Этот репозиторий сначала собирается как **архитектура и документы**, а не как хаотичный набор экранов.

## Главные принципы

- mobile-first
- без обязательной длинной прокрутки на первом экране
- один экран = одна главная задача
- тексты, правила и структура данных хранятся отдельно от вёрстки
- сайт должен легко меняться без поломки
- визуально: тихая дорогая глубина, а не шумный лендинг

## Слои продукта

1. Главная
   - объясняет, что это
   - даёт первый понятный вход

2. Детектор манипуляции
   - главный полезный инструмент
   - анализирует текст на давление, срочность, вину, страх и туман формулировки

3. Досье / архив
   - объясняет механизмы глубже
   - обучает быстрее распознавать их в жизни

Сначала кладём в репозиторий **логику как документы**.
